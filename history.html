Tips & Tricks

  • Tip 1: Use the search fields to filter history by date, title, domain, or parameters.
  • Tip 2: Click on a domain to view all URLs from that domain.
  • Tip 3: The Parameters column shows URL query parameters in a nested table format.
  • Tip 4: Use keyboard shortcut Ctrl+Shift+H to open this extension quickly.
